## üí° Usage

1.  **Select Vendor:** Use the dropdown or dedicated section to choose the active vendor whose details (name, logo, address) will appear on the invoice.
2.  **Enter Client Details:** Input the customer's name, address, and relevant billing information.
3.  **Manage Line Items:**
    * Click "Add Item" to insert a new row.
    * Enter a **Description**, **Quantity**, and **Unit Price**.
    * Watch the totals update automatically.
4.  **Choose Template:** Select one of the 5 available designs from the template selector.
5.  **Generate Invoice:** Click the **"Generate PDF"** or **"Export"** button. The application will render the current view into a clean, A4-sized PDF file.
